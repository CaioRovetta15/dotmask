Download the NN
    * You need to download the desired neural network from their original repo.

    YOLACT and YOLACT++: 
    ```bash
    cd catkin_ws_py3/src/dotmask/nn
    git clone https://github.com/dbolya/yolact.git
    ```

    Mask R-CNN: 
    ```bash
    cd catkin_ws_py3/src/dotmask/nn
    git clone https://github.com/matterport/Mask_RCNN.git
